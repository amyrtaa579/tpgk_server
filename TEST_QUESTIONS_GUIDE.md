# Профориентационный тест — Вопросы и ответы

## Как добавить вопросы через админ-панель

1. Войдите в админ-панель (`/admin`)
2. Перейдите в раздел **"Тесты"** → **"Вопросы теста"**
3. Нажмите **"Добавить вопрос"**
4. Заполните поля:
   - **Текст вопроса**
   - **Варианты ответов** (JSON массив)
   - **Изображение** (опционально, URL)
   - **Документы** (опционально)

## Результат теста

Пользователь получает:
1. **Рекомендацию** — текст с подходящим направлением
2. **Мотивацию** — описание почему подходит эта специальность
3. **Список специальностей** — 1-3 специальности с кодом, названием, сроком обучения и количеством бюджетных мест
